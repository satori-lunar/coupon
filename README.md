# A Book of Moments

A romantic, artistic digital coupon book web app built with Next.js 14, React, Tailwind CSS, and Framer Motion.

## Features

- Beautiful page-flip animations
- Touch and swipe gestures for mobile
- Keyboard navigation (arrow keys)
- Persistent coupon redemption state using localStorage
- Romantic, hand-crafted aesthetic with paper texture
- Responsive design (mobile-first)

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Google Fonts (Dancing Script, Playfair Display, Crimson Text)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment to Vercel

1. Push your code to a Git repository (GitHub, GitLab, or Bitbucket).

2. Import your repository in [Vercel](https://vercel.com).

3. Vercel will automatically detect Next.js and configure the build settings.

4. Deploy!

The app is ready for deployment with no additional configuration needed.

## Customization

To personalize the book:

1. Edit `config/personalization.ts` to add her name (it will appear on the cover)
2. Modify `data/coupons.ts` to change or add coupons
3. Adjust colors in `tailwind.config.js` to match your preferred palette
